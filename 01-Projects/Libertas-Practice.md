## Diary integration
- **Libertas Diary** — lexhosted.com — Google Calendar integration active
- Each hearing generates a calendar event → vault picks up via GCal MCP
- PLAUD device captures hearing transcripts → syncs to matter sub-folder
- Pre-hearing briefs generated automatically 30 minutes before each listing

## Matters

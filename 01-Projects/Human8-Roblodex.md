# Human8 — Roblodex

## Overview
Roblodex is a Roblox game discovery and intelligence platform. Category: "Experience Intelligence." Applies MALIS-style topographical knowledge mapping to the game taxonomy. Three core engines: The Indexer, The Meaning Machine, The Signal Engine.

## Monetisation Architecture — Five Revenue Streams

### 1. Developer Intelligence (B2B SaaS — Primary)
Studio dashboard: audience intelligence, competitor positioning, GDD analysis.
Studio Free £0 / Studio Pro £29/mo / Studio Growth £99/mo / Enterprise Custom.
TAM: 3.5M developers.

### 2. Player Pro (Consumer Freemium — Month 6+)
Enhanced search, personalised alerts, ad-free, extended history analytics.
Player Free £0 / Player Pro £4.99/mo / Annual £39.99/yr.
TAM: 36.7M monthly paying users.

### 3. Sponsored Discovery (Intent-Matched CPC)
Developers bid on semantic categories their game legitimately occupies.
Broad genre CPC £0.15 min / Tag intersection £0.08 min / Game Boost £30/wk.
TAM: 7M active experiences.

### 4. Brand Intelligence + Matchmaking
Reports, sponsored taxonomy features, taste card brand moments, developer × brand matching.
Intelligence report £2.5K–£10K / Branded collection £1.5K–£5K/mo / Taste card £10K–£50K / Matchmaking 10–15%.
Context: 350+ branded Roblox activations in 2025.

### 5. Roblodex Report (Data Media)
Weekly genre intelligence. Sponsored edition £5K–£15K / Institutional sub £200–£500/mo / Data licensing commercial.

## Monetisation Principles

- "Never monetise what feeds the flywheel, only monetise what the flywheel produces"
- First ~6 months: build user base & data moat
- B2B brand intelligence Phase 1, sponsored taxonomy Phase 2
- NEVER sell user data — aggregate and anonymise (kid-dominated platform)

## Market Context (Roblox Q4 2025)

144M DAU (+69% YoY), $1.5B creator payouts, 27% verified 18+ (fastest growing), 18+ spend ~40% more.
